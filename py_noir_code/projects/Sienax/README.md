Sienax
===

SIENA is part of the FSL tools. SIENA is a package for both single-time-point ("cross-sectional") and two-time-point ("longitudinal") analysis of brain change, in particular, the estimation of atrophy (volumetric loss of brain tissue)
# Parameters

### Processing input filtering :

In the ids_to_exec.txt file, shanoir examination ids have to be written. The sienax_json_generator.py will filter all the datasets inside according to the filters written in the file (T1 required for the sienax pipeline)
